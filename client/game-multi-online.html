▶ CLIQUE PARA COMEÇAR

COMO JOGAR ONLINE

1HOST: Digite seu nickname e clique em CRIAR SALA. Um código de 5 letras será gerado.
2Compartilhe o código com o oponente via mensagem, WhatsApp, Discord etc.
3GUEST: Digite seu nickname, cole o código recebido e clique em ENTRAR NA SALA.
4A conexão é peer-to-peer via PeerJS — funciona entre redes Wi-Fi diferentes, 4G, sem configurar nada.
5Quando os dois estiverem conectados, o jogo inicia automaticamente em turnos alternados.
!Em redes corporativas muito restritivas (bloqueio de WebRTC), pode ser necessário usar outra rede.
